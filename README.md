# Ecommerce-Data-Analysis-
Data analysis project using SQL and Python

Ecommerce Sales Data Analysis (SQL & Python)
This project performs data analysis on an E-commerce dataset using Python (Pandas, Matplotlib, Seaborn) and SQL (MySQL). The goal is to derive actionable insights from sales data.

Key Features:
Database Connectivity: Connected MySQL database with Python for data extraction.

Data Analysis:

List unique customer cities.

Calculate order counts for specific years.

Analyze total sales per product category.

Visualization: Used Matplotlib and Seaborn for data representation.

Tools Used:
Languages: Python, SQL

Libraries: Pandas, Matplotlib, Seaborn, MySQL Connector, NumPy

Database: MySQL

How to run:
Ensure you have MySQL installed and the ecommerce database set up.

Update the database credentials in the notebook (db = mysql.connector.connect(...)).

Run the cells in the Jupyter Notebook sequentially.
